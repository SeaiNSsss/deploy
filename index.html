Machine Project

โปรเจกต์เกี่ยวกับเครื่องจักร/ระบบอัตโนมัติ รวบรวมสเปก ไฟล์ และสื่อที่เกี่ยวข้อง ในที่เดียว เข้าถึงข้อมูลล่าสุดได้จากลิงก์ Google Drive ด้านล่าง

เปิดโฟลเดอร์ Drive ดูรายละเอียด

เกี่ยวกับโปรเจกต์

หน้านี้เป็นหน้าเว็ป HTML + CSS เบื้องต้น ออกแบบให้เรียบง่าย อ่านง่าย และรองรับมือถือ ธีมเน้นกลิ่นอายของเครื่องจักรและวิศวกรรม เช่น รูปทรงเฟือง เส้นสาย และโทนสีเมทัลลิก

  • โครงสร้างชัดเจน

    มีส่วน Hero, เกี่ยวกับ, แกลเลอรี, และเอกสารให้เข้าถึงอย่างรวดเร็ว

  • รองรับทุกหน้าจอ

    เลย์เอาต์ปรับตามขนาดหน้าจออัตโนมัติด้วย CSS ที่ตอบสนอง

  • ลิงก์แหล่งข้อมูล

    ปุ่มเข้าถึง Google Drive ของโปรเจกต์โดยตรง
